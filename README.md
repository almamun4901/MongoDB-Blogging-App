# MongoDB Blogging App

This application processes blog transactions from stdin and stores them in MongoDB.

## Requirements

- Python 3.6+
- pymongo
- MongoDB server

## Configuration

Edit the `config.ini` file to set your MongoDB connection details:

```ini
[MongoDB]
host = localhost
port = 27017
database = blog_db
